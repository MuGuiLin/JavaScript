JSON简介

JSON官方标准检测

			一、JSON 是什么？
				JSON的全称是 JavaScritp Object Notation ,即 Javascript 对象标记法。
				JSON是一种轻量级（Light-Weig）、基于文本的（TExt-Based）、可读的（Human-Readable）格式。
			注：JSON的名称中虽然带有JavaScript，但这是指其语法参考了JavaScript对象的，而不是指只能用于JavaScript语言中，其他很多语言都可以用！
			
			JSON数据格式的创始人声称此格式永远不升级，这就表示JSON格式具有较长时间的稳定性。
			
			常用方法：
				1、JSON.parse()方法	//把字符串转化为 JS对象。
				2、JSON.stringify()方法。//把SJ对象转化为 字符串。
			
			二、JSON的语法规则：
			
			注：在JSON中只能用双引号，不能用单引号。
			
				1、数组（Array）用方括号 [] 表示。
				2、对象（Object）用大括号{} 表示。
				3、名称/值对（name/value）键值对组合成数组和对象。
				4、名称（name）置于双引号中，值(value)可存放字符串、数值、布尔值、null、对象和数组等。
				5、并列的数据之间用 逗号 , 分隔开。
			例：
				{"name" : "小穆" , "age" : 25 , "sex" : "男" , "朋友" : ["小钟" , "小张"]}
				
			三、JSON 和 XML
				JSON 常被拿来与 XML 做比较，因为JSON的诞生就是想取代 XML 的意思。
				
			JSON 与 XML相比：JSON的优势如下：
				1、JSON没有结束标签，长度更短，读写更快。
				2、JSON能够直接被Javascript解析器解析。
				3、JSON中可以存放数组。
			
			JSON格式：
				{
					"name" : "小穆",
					"age" : 26,
					"friends" : ["小钟" , "小张" , "小李"]
				}
			
			XML格式：
				
					小穆
					26
					小钟
					小张
					小李
				
				
			四、JSON 的解析 和 生成
				在JavaScript中，有两个方法与此相关：
				
				1、JSON.parse()方法	//把字符串转化为 JS对象。
				2、JSON.stringify()方法。//把SJ对象转化为 字符串
				
			例：
				var str = '{"name" : "小穆" , "age" : 25 , "sex" : "男" , "朋友" : ["小钟" , "小张"]}'; //这是一个字符串！
		
				var obj = JSON.parse(str); //把字符串转化为 JS对象。
				
				console.log(obj);
				alert(obj);
				
				var jsonstr = JSON.stringify(obj); //把SJ对象转化为 字符串
				
				console.log(jsonstr);
				alert(jsonstr);
