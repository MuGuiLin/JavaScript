Brendan Eich 【布兰登·艾奇】 JavaScript的发明人(JavaScript 之父)，1995年任职于网景期间，为网景公司（(Netscape）浏览器 用10天时间就把Javascript设计出来，之后Javascript成为浏览器上应用最广泛的脚本语言之一，目前（2005年至2014年）在Mozilla公司担任首席技术长！


Douglas Crockford 【道格拉斯·克罗克福德】Web 开发领域最知名的技术权威之一，是JSON、JSLint、JSMin和ADSafe的创造者, ECMA JavaScript 2.0 标准化委员会委员，被 JavaScript 之父称为 JavaScript 界的宗师，曾任Yahoo!资深JavaScript架构师，现任PayPal高级JavaScript架构师！
