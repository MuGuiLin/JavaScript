Navigator.sendBeacon()

请求方式与数据格式详解

sendBeacon() 请求方式

navigator.sendBeacon() 方法仅支持 POST 请求，无法指定其他HTTP方法。

支持的数据类型

字符串
FormData
Blob
ArrayBuffer
ArrayBufferView

主要特点

  • 仅支持 POST 请求方式
  • 在页面卸载时可靠地发送数据
  • 不会延迟页面卸载或影响下一导航的加载性能
  • 异步执行，不阻塞页面
  • 适用于分析、诊断和数据收集

局限性

  • 无法自定义请求头 (如 Content-Type)
  • 无法处理服务器响应
  • 无法设置超时或重试机制
  • 数据大小有限制 (通常约64KB)
  • 无法取消已发送的请求

与传统请求方法比较

代码示例

发送字符串数据：

// 发送字符串
const data = 'event=page_view&url=' + encodeURIComponent(location.href);
navigator.sendBeacon('/log', data);

发送JSON数据：

// 发送JSON（作为字符串）
const analyticsData = {
  event: 'click',
  element: 'button',
  timestamp: Date.now()
};
navigator.sendBeacon('/analytics', JSON.stringify(analyticsData));

发送FormData：

// 发送FormData
const formData = new FormData();
formData.append('user', 'user123');
formData.append('action', 'logout');
formData.append('time', Date.now());
navigator.sendBeacon('/track', formData);
