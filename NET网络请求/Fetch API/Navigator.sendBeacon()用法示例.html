Navigator.sendBeacon()

在页面卸载时可靠地发送数据

什么是 sendBeacon？

navigator.sendBeacon() 方法用于通过HTTP将少量数据异步传输到Web服务器。

主要特点

  • 在页面卸载时可靠地发送数据
  • 不会延迟页面卸载或影响下一导航的加载性能
  • 异步执行，不阻塞页面
  • 适用于分析、诊断和数据收集

语法

navigator.sendBeacon(url, data);

参数

url: 数据将要被发送到的网络地址

data: 要发送的数据，可以是ArrayBuffer、ArrayBufferView、Blob、DOMString或FormData

返回值

如果用户代理成功排队传输数据，则返回true，否则返回false。

使用场景

  • 页面关闭前的数据统计
  • 用户行为分析
  • 错误日志记录
  • 会话结束事件

实际应用示例

在页面卸载时发送数据到服务器：

window.addEventListener('beforeunload', function() {
  const data = JSON.stringify({
    event: 'page_unload',
    url: window.location.href,
    timestamp: Date.now()
  });
  
  navigator.sendBeacon('/api/analytics', data);
});

与传统方法的比较

使用 XMLHttpRequest 在 beforeunload 中发送请求可能会阻塞页面卸载，而 sendBeacon 不会。
