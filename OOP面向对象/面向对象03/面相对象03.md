### 组件是什么？

​	 组件 ：数据和方法的简单封装；

​	

### 弹窗功能

- 属性：宽、高、标题、内容；
- 方法：打开 、关闭、渲染、拖拽

- 配置

  ```js
  {
  	width: "30%",
  	height: "250px",
  	title: "测试标题",
  	content: "测试内容",
  	dragable: true, //是否可拖拽
  	maskable: true, //是否有遮罩
  	isCancel:false //是否有取消
  }
  ```
