JS继承

			
			//js中怎么去实现继承的：是采用原型链的概念
		
			//在此之前，你应该了解：构造函数、原型对象、实例对象 这3者之间的关系？
			
			/*
			 * 1、构造函数.prototype == 【原型对象】；
			 * 2、原型对象.constructor == 【构造函数的模板】；
			 * 3、原型对象.isPrototypeOf(实例对象)；//判断实例对象的原型，是不是当前对象
			 * 4、构造函数 和 实例对旬 == 【类 和 实例】；
			 */
			
			/*
			 * 测试：如果让原型对象等于另一个类型的实例，结果会怎么样呢？
			 * 或者：如果让 子类的原型对象 等于 父类的实例，结果会怎么样呢？【这就是传说中 js的继承】
			 * 
			 * 1、此时的原型对象将包含一个指向另一个原型的指针；
			 * Sup的实例对象 和 Sup的原型对象有一个关系【原型对象.isPrototypeOf(实例对象)】
			 * 
			 * 2、相应的另一个原型中也包含着一个指向另一个构造函数的指针；【这就是传说中 js的继承】
			 * 子类原型对象的构造器 变成了 父类的构造器(那么此时，父类有什么属性或方法，子类自然就有了，注：子类有的，父类没有)
			 */
			
			
			//父类构造函数SuP
			function SuP(name , sex)
			{
				this.name = name;
				this.sex = sex;
			}
			
			//父类的原型对象
			SuP.prototype = {
				constructor : SuP,
				sayName : function()
				{
					alert(this.name);
				}
			}
			
			//子类构造函数SuB
			function SuB(age)
			{
				this.age = age;
			}
			
			//alert(SuB.prototype.constructor);//原型对象.constructor == 【构造函数的模板】；
			
			SuB.prototype = new SuP('小穆','男'); //这就是继承：【把父类SuP的实例 赋给 子类SuB的原型对象】
			
			var SuB1 = new SuB(24);
			alert(SuB1.name + SuB1.sex + SuB1.age);
			SuB1.sayName();
