模块化-export-import

        ES5模块化：
            以前是用: SeaJS 或 RequireJS
                * SeaJS是一个遵循CMD规范的JavaScript模块加载框架，可以实现JavaScript的模块化开发及加载机制。
                * RequireJS 是一个JavaScript模块加载器。它非常适合在浏览器中使用，但它也可以用在其他脚本环境，就像 Rhino and Node。使用RequireJS加载模块化脚本将提高代码的加载速度和质量。

        ES6模块化：
            现在是：ES6自带模块化 
                * export 向外暴露或导出模块 export default xxx;
                * import 引入暴露或导出的模块 import {xx, xx} from './xxx.js';

        模块化 只要搞懂：如何定义(导出)模块 和 如何使用(引用)模块 就OK了！！！
